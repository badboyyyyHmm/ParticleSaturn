# 粒子土星

一个基于手势控制的3D粒子土星交互系统，通过摄像头识别手掌动作，实时控制80万粒子组成的土星模型。

## 主要功能
- **80万粒子渲染**：高性能WebGL着色器技术实现的土星粒子系统
- **手势交互控制**：通过手势控制土星缩放、旋转和视角
- **宇宙环境模拟**：包含星空、星云和行星系统的完整宇宙场景
- **实时数据可视化**：显示手部关键点和手势参数

## 快速开始

### 环境要求
- 支持WebGL的现代浏览器（Chrome/Edge推荐）
- 摄像头权限
- 本地服务器环境（无法直接打开HTML文件）

### 启动步骤
1. 下载项目文件
2. 双击运行 `deploy.bat`（Windows）或在终端执行：
   ```bash
   npx serve -p 3000
   ```
3. 系统会自动在浏览器中打开应用
4. 允许摄像头访问权限
5. 将手置于摄像头前开始交互

### 交互说明
- **缩放**：拇指与食指间距离控制土星大小
- **俯仰**：手部上下移动改变观察视角
- **旋转**：手部左右移动控制土星自转
- **注意**：系统追踪手掌平面移动，手腕转动无效

## 项目结构

```
粒子土星/
├── particleSaturn.html    # 主页面（包含所有代码）
├── libs/   # 第三方库（部分MediaPipe手势识别相关文件，还有一部分从网络下载）
│   └─ ···
├── deploy.bat            # Windows部署脚本
└── README.md             # 说明文档
```

### 部署说明
- `deploy.bat`：Windows一键启动脚本，自动安装依赖并启动本地服务器
- 必须通过本地服务器访问（如http://localhost:3000/），直接打开HTML文件会因浏览器安全策略无法加载libs中的文件

## 已知限制

1. **浏览器兼容性**：需支持WebGL 2.0
2. **光线要求**：良好光照条件下手势识别更准确
3. **性能考量**：80万粒子对GPU有一定要求，集成显卡可能卡顿
4. **交互范围**：手部需保持在摄像头视野内，距离1-2米效果最佳
